Reflection

A seemingly basic assignment where one could have imported an old assignment's function into a new file and simply be finished, I saw as an opportunity to try something new, so I made a small little tic tac toe game! It works with many functions and even classes as well. Running in the terminal, two players can play tic tac toe against each other! while programming this, I faced many very unique problems that I have never dealt with before, however, with each new problem, arrived a new and creative solution!

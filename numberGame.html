Reflection

For this assignment, we had to make a simple number guessing game. It was a fun little starter assignment for the programming section of the course, once finished the basic parts, I found myself wanting to add on more and more functionality to the program, however, I had to learn where to draw the line and when the assignment is ready for submission. I had a small list of features I wanted to add on however, after getting it refined to this point, I decided the program is already strong and does not require more add ons, this helped me learn when I should stop adding on, however, I believe I'll have to try more as I tend to have this issue frequently.
